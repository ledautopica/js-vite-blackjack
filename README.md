# js-vite-blackjack
Repositorio de juego BalckJack de JS

Pasos para ejecutar el proyecto:
1. Clonar repositorio
2. Ejecutar 'npm install'
3. Correr el devServer ``npm run dev``
4. Abrir el proyecto en: ``http://localhost:3000``
